Design Process 

Empathize

Empathize is the first step in design process because it is a skill that allows us to understand and share the same feelings that others feel.

Define

Identify the problem or problems experienced by the user by mentioning the problem statement

Ideate

The process of generating a series of ideas based on a particular topic to get creative ideas

Prototype

The results of implementing ideas that are still abstract become real to evaluate ideas that are being designed so that their impact can be felt

Test

The need to test our solutions so that we can improve them. User feedback is priceless—without it, the iterative design process will fail.

About Me

Hello! My name is Ghilman Faza, you can call me Faza. I am a UI/UX Designer. As a UI/UX Designer, I create products according to client needs by paying attention to aspects of beauty and user comfort so that it can attract users and make users comfortable in using the product.

In my daily life, the technologies I usually use to help with my work include Figma, Balsamiq, Maze, Jira. Besides that, I still continue to learn other and new technologies, and share my knowledge with the community.
